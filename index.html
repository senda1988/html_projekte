1. Was sind Backups?

Backups sind einfach Kopien von Daten, die du machst, um deine wichtigen Informationen zu schützen. Wenn etwas schief geht, zum Beispiel ein Fehler oder ein Computerabsturz, kannst du die Daten aus einem Backup wiederherstellen.

2. Warum sollte man Backups machen?

  • Schutz vor Datenverlust: Wenn du deine wichtigen Dateien verlierst, z. B. durch einen Computerfehler oder ein versehentliches Löschen, kannst du sie mit einem Backup einfach wiederherstellen.
  • Einfacher bei Änderungen : Wenn du deinen Computer oder dein Handy zurücksetzt oder eine neue Software installierst, kannst du alle deine Daten aus dem Backup wiederherstellen.
  • Zeit sparen :Backups machen es einfacher und schneller, verlorene Daten wieder zu bekommen.

3. Was macht der Code?

        import shutil
        import datetime

        backup_source = "C:/Users/Benutzer/Dokumente"
        backup_target = "D:/Backup_" + datetime.datetime.now().strftime("%Y%m%d%H%M")

        shutil.copytree(backup_source, backup_target)
        print(f"Backup erfolgreich: {backup_target}")
  1. Importiert die zwei Module:
    • shutil (zum Kopieren von Dateien)
    • datetime (um die aktuelle Zeit zu bekommen)
  2. Ordner zum Sichern festlegen:

    3. Der Quellordner ist der Ordner, den du sichern möchtest. In diesem Fall ist es C:/Users/Benutzer/Dokumente.

  3. Zielordner für das Backup erstellen:

    4. Der Zielordner bekommt einen Namen, der mit Backup_ beginnt, gefolgt vom aktuellen Datum und der Uhrzeit, damit der Ordner immer einen einzigartigen Namen hat. Zum Beispiel könnte der Ordner Backup_202502101200 heißen.

  4. Daten kopieren:

    5. Der Code kopiert den gesamten Inhalt des Dokumente-Ordners in den neuen Zielordner.

  5. Erfolgsmeldung anzeigen:

    6. Am Ende zeigt der Code an, dass das Backup erfolgreich war und sagt dir, wo die Dateien gespeichert wurden.

4. Automatische Ausführung eines Scripts

Um das Script automatisch auszuführen, kann man einen Task Scheduler nutzen:

  • Unter Windows:

    • Windows hat eine eingebaute Funktion, die Programme automatisch startet: Taskplaner (Task Scheduler)

    So geht’s:
    1. Öffne die Aufgabenplanung (Drücke Win + R, tippe taskschd.msc, drücke Enter).
    2. Neue Aufgabe erstellen (Klicke auf „Aufgabe erstellen...“).
    3. Wähle die Uhrzeit:
      • Reiter "Trigger" → "Neu..."
      • Stelle ein: Jeden Tag um 02:00 Uhr starten.
    4. Wähle dein Skript:
      • Reiter "Aktionen" → "Neu..."
      • Wähle dein Python-Skript (backup.py).
  • Unter Linux/macOS: cron jobs

Ein Beispiel für einen cron-Job, der das Script täglich um 2 Uhr ausführt:

0 2 * * * /usr/bin/python3 /path/to/backup.py

Dadurch wird das Script jeden Tag um 02:00 Uhr automatisch ausgeführt.
